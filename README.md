# CSP-lab4-
Для пришвидшення пошуку в задачі CSP використано евристики.
Евристика MRV дозволяє вибирати змінну з мінімальною кількістю допустимих значень, що зменшує кількість тупикових гілок.
Евристика Degree використовується для вибору змінної з найбільшою кількістю обмежень.
Евристика LCV дозволяє обирати значення, яке найменше обмежує можливі значення сусідніх змінних.
